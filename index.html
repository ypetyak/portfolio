Petiton

The idea behind this project is to create an online petition that visitors can sign to make their voice heard on an issue of your choosing. The website starts with the welcome page where you can proceed to registration page or login page. The site uses cookie sessions to figure out if the user is logged in and if true, user will be redirected to the siging page. Similarly if the user tries to access any other page without being logged in, he/she will be redirected to login page.

Password is encrypted with the bcrypt algorithm and stored in our database. The website also stored various user data, such as name, email, address, signature etc. It is also possible to change provided information in Profile section of the website.

The website uses handlebars library to orginize its structure.

The canvas for signing is created through the pure Javascript and allows user to experience signing of a petition as if it was done by hand. Obviously there is an option to delete your petition if you changed your mind or simply unsatisfied with the quality of your signature.

Imageboard

Imagine a site where anybody could go and post an image of their choosing, along with a title and a textual description of that image, and others could go and look at that image and make thoughtful remarks about it. Yes, that's an imageboard.

This website uses a Vue.js framework and is a SPA (single page application). When user upload an image, the website uses the AWS (Amazon Web Services) to store provided images. Additional functionality allows to open every image and comment on it.

Coworking Homes

This website allows user to be the host or participant of the coworking event. The Coworking Homes help people to turn their lonely homes in fun coworking environment. Or join one.

It works on React/Redux framework and utilize Google Maps API to create, find and join events.

Users can create profile for themselves and their homes, after which they can create or join coworking events. Events are created or selected by choosing a day user wants to participate.
